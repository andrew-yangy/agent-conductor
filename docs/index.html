Features

Everything you need to run an AI team

A complete framework for orchestrating autonomous AI agents with real-time visibility into what they're doing and why.

🎮

Pixel-Art Office Simulation

Watch your AI agents move, type, and collaborate in a retro pixel-art office. Each agent has a desk, animations, and real-time status indicators.

🤖

Autonomous Agent Teams

Named C-suite agents -- COO, CTO, CPO, CMO -- each with distinct personalities, domain expertise, and decision-making authority.

📋

Directive Pipeline

Structured triage, planning, building, and review. Weight-adaptive: lightweight tasks run fast, heavyweight gets full challenge and approval cycles.

📊

Live Dashboard

Real-time session tracking, approval workflows, cost insights, and WebSocket-powered state updates. Know exactly what your agents are doing.

Quickstart

Up and running in 60 seconds

Clone the repo, install dependencies, and launch the dashboard. No configuration required.

terminal 
# Clone and install
$ git clone https://github.com/andrew-yangy/gruai.git
$ cd gruai && npm install

# Launch the dashboard + pixel-art office
$ npm run dev

# Open http://localhost:5173

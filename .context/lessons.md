> Lessons about the conductor system itself -- agent behavior, orchestration patterns, skill design.
> Consumer project lessons (DB, ES, infra) live in `.context/lessons.md`.

## Topic Files

Lessons are split by topic so agents read only what's relevant to their role.

| Agent Behavior | `lessons/agent-behavior.md` | All agents (especially Morgan, Sarah) |
| Orchestration | `lessons/orchestration.md` | Morgan (planning), Alex (execution) |
| State Management | `lessons/state-management.md` | Alex (execution), Sarah (review), engineers |

## How to Use

**For agents:** Read the topic file(s) relevant to your role. Don't read all of them unless you need cross-cutting context.

**For new lessons:** Add to the appropriate topic file. If a lesson spans topics, put it in the primary topic and cross-reference.

**For consolidation (every 10th directive):** Re-read all reports, update topic files, remove one-off entries that haven't recurred.
